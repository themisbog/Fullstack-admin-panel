# Fullstack Admin Dashboard (In Progress 🚧)

## Overview
This is a comprehensive full-stack application that transforms a static Admin Dashboard into a dynamic system. It bridges a modern UI (Odin Project) with a robust Java Backend (University of Macedonia studies).

The project demonstrates my ability to handle end-to-end development, from UI design to REST API implementation and database persistence.

## Tech Stack
* **Frontend:** HTML5, CSS3 (Grid/Flexbox), Vanilla JavaScript (ES6+).
* **Backend:** Java with Spring Boot 3.x.
* **Database:** H2 In-Memory Database.
* **API:** RESTful architecture with JSON exchange.
