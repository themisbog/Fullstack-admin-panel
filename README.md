# Fullstack Admin Dashboard (In Progress 🚧)

## Overview
This is a comprehensive full-stack application that transforms a static Admin Dashboard into a dynamic system. It bridges a modern UI (Odin Project) with a robust Java Backend (University of Macedonia studies).

The project demonstrates my ability to handle end-to-end development, from UI design to REST API implementation and database persistence.

## Tech Stack
* **Frontend:** HTML5, CSS3 (Grid/Flexbox), Vanilla JavaScript (ES6+).
* **Backend:** Java with Spring Boot 3.x.
* **Database:** H2 In-Memory Database.
* **API:** RESTful architecture with JSON exchange.

## Features (Developing)
- [x] **Dynamic Rendering:** JavaScript Fetch API retrieves project data from the backend in real-time.
- [x] **Interactive UI:** Modal-based form implementation for adding new projects without page refreshes.
- [x] **Persistent Storage:** Spring Data JPA integration for managing project data in the database.
- [x] **Responsive Layout:** Professional dashboard design inspired by The Odin Project, optimized with CSS Grid.
- [x] **Data Seeding:** Automated database initialization with realistic project samples via a custom `DataLoader`.

I am building this to master:
